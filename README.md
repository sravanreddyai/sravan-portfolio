# Sravan Gatla — Portfolio Website

A clean, responsive, single-page portfolio built with plain HTML/CSS/JS.

## Quick start (local)

Just open `index.html` in your browser.

## Deploy on GitHub Pages

1. Create a new repo (e.g., `sravan-portfolio`)
2. Upload all files from this folder
3. In GitHub: **Settings → Pages → Build and deployment**
   - Source: Deploy from a branch
   - Branch: `main` / root
4. Your site will be live at `https://<username>.github.io/sravan-portfolio/`

## Customize

- Replace the photo in `assets/` and update the filename in `index.html`
- Update the Featured Projects section with public GitHub links if you have them
- Edit text content directly in `index.html`
- Modify colors and styling in `styles.css`
- Update contact form email in `script.js` and `index.html`

## Features

- Responsive design (mobile-friendly)
- Modern dark theme with gradient backgrounds
- Mobile navigation toggle
- Contact form with mailto integration
- Accessible HTML structure
- Fast loading (no frameworks or dependencies)
- Customizable color scheme using CSS variables
